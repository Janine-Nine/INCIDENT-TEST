# 🚨 Incident Management API

Sistema Full Stack para gerenciamento e análise de incidentes técnicos, desenvolvido com Java Spring Boot, PostgreSQL e arquitetura RESTful.

---

# 📌 Objetivo

Este projeto foi desenvolvido como teste técnico para vaga Full Stack Java + Angular/React, contemplando:

- Desenvolvimento backend
- API REST
- Persistência de dados
- Tratamento de exceções
- Logs
- Monitoramento
- Docker
- Testes automatizados
- Documentação técnica

---

# 🛠 Tecnologias

## Backend
- Java 17
- Spring Boot 3
- Spring Security
- Spring Data JPA
- JWT Authentication
- PostgreSQL
- Swagger/OpenAPI
- Lombok

## DevOps
- Docker
- Docker Compose
- Render

## Testes
- JUnit 5
- Mockito
- MockMvc
